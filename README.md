# EMOVE

Emove est un site de location de véhicule électrique.

##Pré requis
docker, docker-compose

## Installation

```bash
git clone 'repo nom'
```
```bash
cd projet_2eme_annee/
```
```bash
docker-compose up -d
```

## Adresse Locale pour accéder à la BDD et au site 

http://localhost:8080/ (BDD)
http://localhost:5000/ (PORT)
